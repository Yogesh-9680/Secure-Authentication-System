# Secure Authentication System

A simple Secure Authentication System developed in Python for learning the basics of user authentication. This project demonstrates how users can register, log in, and verify credentials using a straightforward authentication process.

## Features

* User Registration
* User Login
* Username and Password Verification
* Basic Input Validation
* Simple Menu-Driven Interface
* Beginner-Friendly Code
* Easy to Run in Online Python Compilers

## Technologies Used

* Python 3

## Purpose

This project is created for educational purposes to understand the fundamentals of authentication systems using Python. It is designed to be simple, easy to read, and executable on online Python compilers such as Programiz.
